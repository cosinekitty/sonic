This archive contains source code for the Sonic/C++ translator.
This source code should be easily portable to most operating systems.
You will need a C++ compiler which supports exceptions to compile this.
No templates are used, so don't worry about that.

For more information about the Sonic programming language, visit the
Sonic home page at:

http://www.intersrv.com/~dcross/sonic/
